## Overview
Prueba técnica para programador backend realizada por <b>Antonio Alonso</b> para la empresa <b>quick.com.co</b>, este desafío tiene como fin evaluar la capacidad de un desarrollador backend para entender y llevar a cabo los requisitos propuestos. En este documento, se plantea un escenario que le permitirá mostrar sus habilidades en desarrollo de software, incentivar su creatividad y promover la solución de problemas. Observaremos tanto el proceso como el resultado final.

## Getting Started

### Prerequisites

- [Node.js 18.XX](https://nodejs.org/)
- [Serverless Framework CLI](https://www.serverless.com/framework/docs/getting-started/)
- [AWS CLI](https://docs.aws.amazon.com/cli/latest/userguide/cli-chap-install.html)
- [Nest JS 10.XX](https://nestjs.com/)

### Installing

1. Clone the repo
   ```sh
   git clone https://github.com/AIMtonio/back-api-prueba-tecnica
   ```
2. Install NPM packages
   ```
   npm install
   ```

## Local Usage
1. Run locally:

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
